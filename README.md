# B站视频解析下载器

一个基于Web的B站视频解析和下载工具，支持输入视频链接或BV号，解析视频信息并下载合并后的MP4格式视频。

## 功能特性

- 🎬 支持B站视频链接和BV号解析
- 🎥 多画质选择（360P到4K）
- 🔄 自动合并视频和音频流
- 📱 响应式设计，支持移动端
- ⚡ 实时下载进度显示
- 🎨 现代化UI界面

## 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS (UI框架)
- FFmpeg.wasm (视频处理)
- B站开放API

## 使用方法

1. 在输入框中输入B站视频链接或BV号
2. 点击"解析视频"按钮
3. 选择想要的画质
4. 点击"下载合并后的MP4视频"按钮
5. 等待下载和合并完成
